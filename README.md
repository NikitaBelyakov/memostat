# memostat
МЕМОСТАТ — система анализа жизненного цикла интернет-мемов. Программа отслеживает популярность мемов во времени, определяет их текущую стадию (от зарождения до смерти).

ЦЕЛЬ
Разработать технологию автоматического обнаружения, классификации и прогнозирования жизненного цикла интернет-мемов на основе анализа их популярности в интернете.

ЗАДАЧИ
1. Сбор данных из RSS Google Trends
2. Обнаружение пика популярности
3. Классификация 7 стадий жизненного цикла: 🟣 Эмбрион (только зарождается), 🔵 Рост (активный рост), 🟢 Взлёт (приближение к пику), 🟡 Пик (максимальная популярность), 🟠 Затухание (спад), 🔴 Стагнация (низкий устойчивый уровень), ⚫ Смерть (ниже порога)
4. Определение 4 типов пост-пиковых траекторий: 📉 Гладкий спад (монотонное убывание), 📊 Колебательный спад (спады с повторными пиками), 📈 Плато (выход на устойчивый уровень), 🚀 Устойчивый рост (нет явного спада)
5. Поиск и добавление новых мемов
6. Визуализация графиков
7. Генерация еженедельных отчётов
